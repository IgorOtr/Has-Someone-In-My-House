Usar minha câmera

Transforma a webcam deste dispositivo numa câmera do monitor, enquanto esta aba estiver aberta.

Voltar ao painel
Ao clicar em "Ativar câmera", o navegador vai pedir permissão para usar sua webcam. O vídeo é enviado a este servidor só enquanto a sessão estiver ativa, para detectar a presença de pessoas; imagens só são salvas quando uma detecção é confirmada. Feche esta aba ou clique em "Encerrar" para parar a qualquer momento.
